# Evaluation Data Usage (Examiner-Safe)

## Data Sources

- **`ausgrid_p2p_energy_dataset.csv`**: Time-series demand and PV (kW) per agent, derived from or aligned with real smart-meter style data (e.g. Ausgrid Solar Home Dataset). Used for **evaluation** when the path is provided (e.g. `python evaluation/evaluate_episode.py --data evaluation/ausgrid_p2p_energy_dataset.csv`).
- **`test_day_profile.csv`** (project root): **Synthetic** profile generated by `generate_data.py`. Used by default in **training** and as fallback in evaluation if the above file is not specified.

## Replay Mechanism

- Data is replayed **sequentially** (no shuffling). Each row corresponds to one timestep (1 hour). The environment indexes by `current_idx` and reads `agent_{i}_demand_kw`, `agent_{i}_pv_kw`.
- For **deterministic evaluation**, use `random_start_day=False` so replay starts at row 0.
